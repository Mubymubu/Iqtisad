# **App Name**: Iqtisad: Behavioral Economics Simulator

## Core Features:

- Home Page: Introduce the simulation, target audience, and level overviews with a 'Start Tutorial' CTA.
- Tutorial System: A guided tutorial to teach users about market mechanics and emotional responses, collecting user feelings for later reflection.
- Core Concepts Database: A set of definitions and explanations of core behavioral economics concepts that will affect decisionmaking. It acts as a tool for informing users how they will make economic decisions.
- Level Simulations (I-III): Simulate stock markets, venture capital, and cryptocurrency trading environments with increasing difficulty and complexity. Each level tracks performance and user decisions.
- Debrief System: Mandatory debriefs at the end of each level that ask the user reflective questions, providing insights into their decision-making biases.
- Strategy Guide: Provide actionable strategies that users can utilize for reflection on their economic biases.
- Contact Form: A portal to get user feedback for iterative improvements on the design.

## Style Guidelines:

- Primary color: Calm blue (#5DADE2) to invoke a sense of trust and stability, suitable for an educational platform.
- Background color: Very light blue (#EBF4FA), providing a clean and unobtrusive backdrop.
- Accent color: Soft green (#A3E4D7), used sparingly to highlight important actions and information.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for both headlines and body text.
- Use clean, simple icons to represent different financial concepts and biases.
- Maintain a minimalistic layout to reduce clutter and cognitive overload.
- Subtle transitions and feedback animations to enhance user experience without distraction.
